Installation
============================


.. code-block:: shell

   # install pdm
   python -m pip install pdm

   # config pdm mirror
   pdm config pypi.url https://pypi.tuna.tsinghua.edu.cn/simple

   # install mentabotix
   pdm add mentabotix

   # or you can install unstable version
   # pdm add mentabotix --pre
